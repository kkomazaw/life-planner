# Life Planner - 家族資産管理・ライフプランシミュレーター

家族の資産ポートフォリオを月単位で記録し、ライフプランにおける出費の実績や計画も記録でき、将来の資産推移をシミュレーションできる家計簿アプリケーション。

## 特徴

- **資産ポートフォリオ管理**: 現金・投資・不動産・その他の資産を月次で記録
- **収支管理**: 項目別に月次で収入・支出を記録
- **ライフイベント計画**: 進学、住宅購入などの大きなイベントを設定
- **将来シミュレーション**: 運用利回りやインフレ率を考慮した資産推移の予測
- **データ可視化**: グラフやダッシュボードで資産状況を一目で把握
- **エクスポート機能**: CSV/Excel/PDFでデータをエクスポート
- **ローカルデータ**: すべてのデータはブラウザ内に保存（プライバシー保護）

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **ビルドツール**: Vite
- **状態管理**: Zustand
- **スタイリング**: Tailwind CSS
- **データベース**: IndexedDB (Dexie.js)
- **グラフ**: Recharts
- **日付処理**: date-fns
- **バリデーション**: Zod

## 開発環境のセットアップ

### 必要要件

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ブラウザで開く
# http://localhost:5173
```

### その他のコマンド

```bash
# ビルド
npm run build

# プレビュー
npm run preview

# Lint
npm run lint

# テスト
npm run test
```
